  <action>
Create `apps/backend/src/routes/profile.ts` with a single `GET /:username` handler. No requireAuth — this endpoint is fully public.

Handler logic:
1. Extract `username` from `req.params`.
2. Look up user by `users.username` (NOT by id) — select `{ id, username, balance, totalWagered, totalProfit }` only. Never return email, passwordHash, tokenVersion, isBanned.
3. If no user found, `res.status(404).json({ error: 'User not found' })` and return.
4. Run four sub-queries in parallel with `Promise.all`:
   - **rankRow**: `rank() OVER (ORDER BY ${users.balance} DESC)` window function, filtered to this user's id (same pattern as leaderboard.ts)
   - **gamesRow**: `count(*)` from `gameLogs` WHERE `userId = user.id`
   - **balanceHistory**: `SELECT createdAt, balanceAfter FROM gameLogs WHERE userId = user.id ORDER BY createdAt ASC` — returns one point per game round
   - **wageredPerDay**: `SELECT DATE_TRUNC('day', createdAt)::date::text AS date, SUM(betAmount) AS wagered FROM gameLogs WHERE userId = user.id GROUP BY DATE_TRUNC('day', createdAt) ORDER BY date ASC` — use Drizzle `sql` template with `.mapWith(Number)` on the sum

5. Return `res.json({...})` with this exact shape:
```typescript
{
  username: user.username,
  balance: user.balance,
  balanceRank: rankRow[0]?.rank ?? 1,
  totalWagered: user.totalWagered,
  totalProfit: user.totalProfit,
  gamesPlayed: Number(gamesRow[0]?.total ?? 0),
  balanceHistory: balanceHistory.map(r => ({ x: r.x.toISOString(), y: r.y })),
  wageredPerDay,
}
```

Note on `balanceHistory.map`: `r.x` is a `Date` object from Drizzle — call `.toISOString()` to produce the ISO string the frontend's Recharts XAxis expects.

Imports needed: `Router`, `type IRouter` from express; `db` from `../db/index.js`; `users`, `gameLogs` from `../db/schema.js`; `eq`, `sql`, `asc`, `count` from `drizzle-orm`.

Export: `export const profileRouter: IRouter = Router();`
